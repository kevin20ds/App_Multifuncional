# React Native IMC + Todo App (Expo-friendly)

Estrutura simples de app em React Native conforme solicitado:
- Tela de Login
- Tela de Criação de Conta
- Tela Inicial com 2 cartões: IMC e Lista de Tarefas
- Tela de Gerenciamento de Conta

## Como usar
1. Instale o Expo: `npm install -g expo-cli` (ou use `npx expo`).
2. No diretório do projeto, rode `npm install`.
3. Rode `npm start` ou `expo start`.
4. Abra no emulador ou em um dispositivo usando o app Expo Go.

> Observação: este projeto evita dependências de navegação externas (React Navigation) e usa navegação simples por estado.
